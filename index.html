AI & Machine Learning

I have a strong background in AI and Machine Learning, with hands-on experience in computer vision, NLP, and data science. I’ve worked with frameworks like PyTorch, TensorFlow, and Hugging Face, developing models for image segmentation, language translation, and fraud detection. Web Design

I build modern, responsive websites using React.js, Three.js, Vue.js, and JavaScript. I also explore 3D web development, using Blender and Three.js to create immersive experiences. I enjoy working with data, transforming raw numbers into meaningful insights. Using tools like Pandas, NumPy, Matplotlib, Seaborn and PowerBI , I analyze datasets and build predictive models. My California Housing Price Analysis project is a great example of using machine learning techniques to understand and forecast market trends.
